There a hundreds of jQuery and pure JavaScript realisations of tooltip functionality. I'd like to offer pure CSS3 variant. Because in CSS3 there are such pretty stuff like animations, transition and vector graphics. Also will be available the graceful degradation for older browsers.

How does it is work?

The idea

When hover your mouse on an element, somewhere nearby small information box will appear.

Realisation

First of all we need to define the structure. There will be an element which contains it's tooltip.


<span class="with-tooltip">
  <span class="tooltip">Tooltip text</span>
  Other HTML here
</span>

Next we need to define styles to this markup. Container should have position relative. So we can position with position:absolute our tooltip as we want. 

.with-tooltip {
  position:relative;
}

Now we need add position styles for tooltip box itself: 

.with-tooltip .tooltip {
  /* nowrap will always show words in one line */
  white-space:nowrap;

  /* to be sure that tooltip is always block-level element */
  display:block;

  /* By default it should be hidden */
  visibility: hidden;
  opacity:0;

  /* Make position absolute */
  position:absolute;
  top:-3em;

  /* just styling can be changed for your needs */
  min-width:200px;
  color:#333;
  -webkit-border-radius:3px;
     -moz-border-radius:3px;
      -ms-border-radius:3px;
          border-radius:3px;
  /* Add transtions for a smoother effect. */
  -webkit-transition: all 0.2s linear 0.1s;
     -moz-transition: all 0.2s linear 0.1s;
      -ms-transition: all 0.2s linear 0.1s;
       -o-transition: all 0.2s linear 0.1s;
          transition: all 0.2s linear 0.1s;
  background: #ffc;
  padding: 5px;
  border:1px solid #ccc;
  -webkit-box-shadow: 0px 0px 4px 0px rgba(0, 0, 0, 0.25);
  box-shadow: 0px 0px 4px 0px rgba(0, 0, 0, 0.25);
}

At last we need to add hover state for our tooltip. Please note that we use active state as well, because now every device has the mouse. 

.with-tooltip:hover .tooltip,
.with-tooltip:active .tooltip {
  opacity: 1;
  visibility: visible;
  top:-2.2em;
}

  top: 100%;
  border: solid transparent;
  content: " ";
  position: absolute;
  pointer-events: none;

.with-tooltip .tooltip:after {
  border-color: rgba(255, 255, 204, 0);
  border-top-color: #ffc;
  border-width: 5px;
  left: 15%;
  margin-left: -5px;
}

  border-color: rgba(51, 51, 51, 0);
  border-top-color: #ccc;
  border-width: 6px;

}
